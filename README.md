# vue-demo

This template should help get you started developing with Vue 3 in Vite.

## 環境建置

- node > 14.17
- npm > 6

```sh
npm install
```

### 起local測試

```sh
npm run dev
```

### 打包成靜態檔案(prod)

```sh
npm run build
```

### 打包後試跑(prod)

```sh
npm run preview
```
